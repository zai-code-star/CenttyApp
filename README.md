# CenttyApp ‚Äì Property Mobile Application

**CenttyApp** adalah aplikasi mobile yang dirancang untuk memudahkan pengguna dalam mencari properti serta berkomunikasi langsung dengan agen **Central Property Purwokerto**.  
Aplikasi ini mengutamakan tampilan modern, kemudahan navigasi, dan proses pencarian properti yang efisien.

CenttyApp membantu pengguna menemukan properti yang sesuai sekaligus mempermudah agen dalam melayani calon pelanggan secara digital.
